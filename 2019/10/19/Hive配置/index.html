Hive

发表于 2019-10-19 更新于 2019-10-20

Command

交互方式一 : (Hive交互Shell)

  • [root@node01 bin]# ./hive

交互方式二 : (Hive JDBC服务)

  • [root@node01 bin]# ./hive –service hiveserver2 (前台启动)
  • [root@node01 bin]# nohup bin/hive –service hiveserver2 & (后台启动)
  • [root@node01 bin]# ./beeline
  • beeline> !connect jdbc:hive2://Xxx:10000
  • 注意：如果使用Beeline方式连接HiveServer2，一定要保证Hive在Mysql当中的元数据库已经创建成功，不然就会拒绝连接

交互方式三 : (Hive命令)

  • [root@node01 bin]# ./hive -e “命令”
  • [root@node01 bin]# ./hive -f “文本”
